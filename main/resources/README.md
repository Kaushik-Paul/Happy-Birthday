# Resources Folder

This folder contains media assets for the Happy Birthday template.

## Required Audio Files

For the full experience with sound effects, add the following audio files:

1. **pop.mp3** - Balloon pop sound effect (used in the balloon mini-game)
2. **magic.mp3** - Magical/sparkle sound effect (transitions and surprises)
3. **candle.mp3** - Candle blow-out sound (triggered when all flames are extinguished)
4. **birthday-music.mp3** - Background birthday music (loops during the experience)

## Where to Get Free Sound Effects

You can download free sound effects from:
- [Freesound.org](https://freesound.org)
- [Mixkit.co](https://mixkit.co/free-sound-effects/)
- [Zapsplat.com](https://www.zapsplat.com)

## Note

The template will work without these audio files - sounds are optional and will gracefully fail if not present.
